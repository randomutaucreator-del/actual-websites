📋 How to use your homepage

After downloading, set it as your browser's startup page.

🟠 Chrome:
Settings → On startup → Open a specific page → Add a new page → file:///path/to/your-homepage.html
🔵 Firefox:
Settings → Home → Homepage and new windows → Custom URLs → paste your file path
🌐 Edge:
Settings → Start, home, and new tabs → Open these pages → Add a new page → paste file path
